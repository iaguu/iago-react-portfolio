# Iago — Portfolio (React + Vite)

## Como rodar
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Estrutura
- `src/pages` páginas principais
- `src/components` componentes reutilizáveis
- `src/data/projects.js` dados dos projetos (edite aqui para alimentar a UI)
- `src/styles/global.scss` tema + tokens + componentes base
